# Privacy Policy

## Data Collection

EU Lens collects minimal data necessary for core functionality:

- **Content for Analysis**: Text and URLs submitted for misinformation analysis (processed ephemerally)
- **Accessibility Preferences**: Stored locally in your browser — never transmitted to servers
- **Theme Preference**: Stored locally in your browser
- **Usage Analytics**: Anonymous, aggregate metrics only

## Data We Do NOT Collect

- Personal identification information
- Passwords or authentication credentials
- Browsing history (beyond explicitly submitted URLs)
- Location data
- Biometric data
- Sensitive personal data

## Data Storage

- Accessibility and theme preferences: Browser localStorage only
- Analysis history: Browser memory (session only)
- API requests: No permanent storage of submitted content

## Data Sharing

EU Lens does not sell, trade, or share personal data with third parties.

## User Rights

Under GDPR, users have the right to:
- Access their data
- Rectify inaccurate data
- Erase their data ("right to be forgotten")
- Restrict processing
- Data portability
- Object to processing

## Contact

For privacy-related inquiries: privacy@eulens.app

## Updates

This policy may be updated. Users will be notified of material changes.
